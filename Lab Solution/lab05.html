Concept of WWW

  • It Stands for World Wide Web.
  • It is basically a system of internet servers that support formatted documents.
  • User access the World Wide Web facilities via called a web browser

HTTP Protocol

  1. HTTP stand for Hypertext Transfer Protocol.
  2. HTTP is based on the client-server architecture model and a stateless request/reponse protocol
  3. 404 FIle Not Found is a common HTTP status code.
